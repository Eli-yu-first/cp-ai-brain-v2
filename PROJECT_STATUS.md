**优势**

**步骤**

**优势**

**步骤**
```bash

cd cp-ai-brain-v2
pnpm install
```

## 已知限制和注意事项

### 当前限制

1. **webdev 工具兼容性**
   - 原因：webdev 工具系统内部状态不一致
   - 影响：无法通过 webdev 工具进行检查点操作
   - 解决方案：使用 GitHub 仓库和手动部署流程

2. **数据库初始化**
   - 需要手动执行迁移脚本
   - 建议在部署前完成

3. **本地认证限制**
   - 仅支持单个管理员账户
   - 适合开发和测试环境
   - 生产环境建议使用 Manus OAuth

### 安全建议

1. **必须修改默认密码**
   ```env
   LOCAL_ADMIN_PASSWORD=<strong-password>
   ```

2. **生成 JWT 密钥**
   ```bash
   openssl rand -base64 32
   ```

3. **启用 HTTPS**
   - 在生产环境中必须使用 HTTPS
   - Manus 平台自动提供 SSL/TLS

4. **定期更新依赖**
   ```bash
   pnpm update
   ```
